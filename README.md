# NOAA Storm Events Data Preparation and Analysis

## Overview

This project uses NOAA Storm Events data to practice data cleaning, preparation, analysis, and visualization.

The first phase focuses on historical data from:

- 1995
- 2005
- 2015

These years were selected as initial comparison points before expanding the project to include additional years and longer-term trends.

## Project Goals

- Review and clean raw NOAA Storm Events data
- Identify inconsistent formats and missing values
- Prepare the data for SQL and Power BI analysis
- Compare storm and tornado activity across selected years
- Explore geographic and historical patterns in severe weather

## Current Status

This project is currently in the data preparation phase.

Current work includes:

- Reviewing the original NOAA datasets
- Cleaning and standardizing selected fields
- Comparing differences in structure across years
- Preparing files for future analysis
- Documenting the cleaning process
